# Full Stack Open Part 1 Exercises
By Thanh Toan Tu

Comprised of 3 small projects: `courseinfo`, `unicafe` and `anecdotes`.

## Repository Structure
```
full-stack-open-part1
├── courseinfo
├── unicafe
└── anecdotes
```

## Usage

1. Navigate to the desired project folder:  
`cd [folder-name]`
1. Install NPM:  
`npm install`
1. Run the project:  
`npm run dev`

## Technologies
- JavaScript
- React
- Vite
